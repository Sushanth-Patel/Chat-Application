## ✨ Overview

A lightweight real-time chat application built with a focus on **clarity, responsiveness, and scalable data flow**.

- OTP-based authentication  
- Instant messaging (Firestore)  
- User discovery  
- Push notifications (FCM)

---

## 🧠 Architecture
UI → Adapters → Models → Firebase

- Real-time sync using snapshot listeners  
- Chatroom-based design (prevents duplication)  
- Structured for easy migration to MVVM  

---

## ⚡ Key Capabilities

- Phone authentication (OTP)
- Real-time messaging (Firestore listeners)
- User discovery (search)
- Recent conversations
- Push notifications (FCM)
